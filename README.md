Chapter2 전체 내용(p.47~p.62)
- 기본적인 퍼셉트론 개념
- 퍼셉트론은 크게 입력값, 가중치, 편향 3가지 기본 요소로 구성
- 기본 퍼셉트론을 이용한 AND 게이트, OR 게이트, NAND 게이트 구현
- 퍼셉트론을 여러개 쌓아서 다층 퍼셉트론을 구현하면 복잡한(비선형) 구조 구현 가능
- 다층 퍼셉트론을 이용한 XOR 게이트 구현
